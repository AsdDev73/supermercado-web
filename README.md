# supermercado-web
Los objetivos de este trabajo de enfoque son los siguientes:
● Desarrollar un diseño atractivo y funcional para un supermercado.
● Aplicar principios de diseño visual, incluyendo la elección de colores y
tipografías.
● Crear un mockup previo al desarrollo.
● Presentar el código HTML y CSS de la web.
● Reflexionar sobre el proceso de diseño y su aplicabilidad profesional.
